## Quick Reference: Time Blocking Rules

1. **Big Rocks First** - Schedule important tasks before reacting to others
2. **Batch Similar Tasks** - Group calls, emails, admin together
3. **Buffer Time** - Leave 25% unscheduled for unexpected items
4. **Energy Match** - Hard tasks during peak energy
5. **Protect Deep Work** - 2-4 hour blocks for creative/strategic work
6. **Transition Time** - 15 min between meetings
7. **End Time** - Set a daily shutdown time and honor it

## Weekly Planning Checklist

Use this checklist every week:

### Before Planning
- [ ] Clear workspace
- [ ] Have calendar open
- [ ] Review last week's notes
- [ ] Check current Impact Project

### During Planning
- [ ] Complete weekly review
- [ ] Set 8-10 weekly targets
- [ ] Identify ONE Thing
- [ ] Block big rocks on calendar
- [ ] Plan daily MINs
- [ ] Check Value time budget
- [ ] Set weekly boundaries

### After Planning
- [ ] Share plan with accountability partner
- [ ] Post weekly targets somewhere visible
- [ ] Prepare for tomorrow (clothes, bag, etc.)

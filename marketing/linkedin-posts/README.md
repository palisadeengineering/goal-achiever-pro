# LinkedIn Post Graphics

Single-image graphics optimized for LinkedIn's feed format (1200x627px).

## Structure

```
linkedin-posts/
├── styles.css          # Shared styles for all LinkedIn graphics
├── week1/              # Vision & Goals theme
├── week2/              # Time & Energy theme
├── week3/              # Routines & Systems theme
└── week4/              # Leverage & Scale theme
```

## Dimensions

- **Width:** 1200px
- **Height:** 627px
- **Aspect Ratio:** 1.91:1

## How to Use

1. Open any `.html` file in a browser
2. Take a screenshot or use a browser extension to capture
3. Export as PNG or JPG for LinkedIn

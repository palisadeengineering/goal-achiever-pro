# X (Twitter) Content System

Complete X/Twitter content library with threads, single posts, and viral hooks.

## Contents

| File | Description |
|------|-------------|
| `threads/` | Full thread templates (10-15 tweets each) |
| `daily-posts.md` | 30 days of single-post content |
| `viral-hooks.md` | 50+ hook formulas that drive engagement |
| `reply-templates.md` | Templates for engaging with audience |

## Posting Strategy

### Optimal Times (EST)
- **Weekdays:** 8 AM, 12 PM, 5 PM
- **Weekends:** 9 AM, 1 PM

### Content Mix
- 40% Value threads (education)
- 30% Single insights (quick wins)
- 20% Engagement (polls, questions)
- 10% Promotion (app, lead magnets)

### Thread Frequency
- 2-3 threads per week
- Best days: Tuesday, Wednesday, Thursday
- Best time: 8-9 AM EST

## Thread Structure

```
1. Hook (pattern interrupt)
2. Promise (what they'll learn)
3. Body (5-10 value tweets)
4. Summary (recap key points)
5. CTA (follow, retweet, link)
```

## Character Limits

- Single tweet: 280 characters
- Thread tweet: Keep under 250 for readability
- First tweet: Front-load value, no links

## Hashtag Strategy

Use 1-2 hashtags max per tweet:
- #productivity
- #timemanagement
- #goals
- #entrepreneurship
- #buildinpublic
